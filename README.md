# Flutter Fundamentals Demo

## StatelessWidget vs StatefulWidget

StatelessWidget is used for static UI that does not change.

StatefulWidget is used when the UI needs to update dynamically based on user interaction.

## Widget Tree

Flutter builds UI using widgets arranged in a tree structure.

When state changes, Flutter rebuilds only affected widgets which ensures high performance.

## Why Dart for Flutter

Dart provides:

- object oriented programming
- async programming
- strong typing
- fast compilation

These features make Flutter apps fast and scalable.
